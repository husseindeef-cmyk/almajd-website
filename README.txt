# موقع المجد للتوريدات العمومية (قالب جاهز)

## التعديل السريع
- رقم الواتساب: افتح `index.html` وابحث عن `businessNumber` واستبدل الرقم (بدون +).
- رقم الهاتف: عدّل رابط `tel:` والنص الظاهر تحت قسم "تواصل معنا".
- العنوان والخريطة: استبدل نص العنوان، وضع رابط خريطة Google في وسم `<iframe>`.
- الشعار: ضع ملف شعار PNG داخل مربع "ضع شعار المجد هنا" أو استبدله بصورة فعلية.
- المنتجات والخدمات: عدّل أقسام "الخدمات" و"منتجات مختارة" حسب حاجتك.

## النشر
- ارفع المجلد على أي استضافة (Netlify, Vercel, GitHub Pages) أو على الاستضافة الخاصة بك.
- الملف الوحيد المطلوب هو `index.html` (لا يوجد باك-إند في هذا القالب).

بالتوفيق!
